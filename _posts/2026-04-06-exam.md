# 2025년 2회 사무자동화산업기사 필기 기출문제 정리

이번 글에서는 2025년 2회 사무자동화산업기사 필기 기출문제를 과목별로 정리했다.
단순히 정답만 나열하기보다, 반복해서 등장하는 핵심 개념을 함께 확인할 수 있도록 구성했다.

## 1과목: 사무자동화 시스템

### 핵심 정리

* 그룹웨어: 정보공유, 업무흐름 관리, 의사결정 지원
* 데이터베이스 시스템: 데이터 공유, 중복 감소, 무결성 유지
* RAID: 디스크 장애 대비 및 안정성 향상
* DSS: 의사결정 지원 시스템
* 클래스(Class): 객체들의 집합
* POP3: 메일을 내려받는 프로토콜

### 문제와 정답

1. 그룹웨어의 기능과 거리가 먼 것은? → **2. 이미지 편집 기능**
2. 데이터베이스 시스템에 대한 설명으로 옳지 않은 것은? → **4. 파일이 파괴되었을 때 복구가 빠르고 완벽하다**
3. 인터럽트 발생 시 현재 프로그램 상태가 저장되는 레지스터는? → **1. PSW**
4. 자동화의 체계 중 사무관리의 자동화가 아닌 것은? → **4. 프로그램의 작성 자동화**
5. 데이터 안정성을 높이는 디스크 기술은? → **3. RAID**
6. 의사결정에 필요한 정보를 검색·분석·출력하는 시스템은? → **2. DSS**
7. 객체지향형 데이터베이스에서 객체들의 모임은? → **2. 클래스(Class)**
8. 메모리에 대한 설명으로 가장 거리가 먼 것은? → **1. RAM은 휘발성이 없다**
9. DBMS 기능으로 옳지 않은 것은? → **2. 탐색 기능**
10. 팩시밀리 특징으로 옳지 않은 것은? → **2. 한 번에 한 명의 수신자에게만 보낼 수 있다**
11. 사무자동화의 특징으로 가장 거리가 먼 것은? → **4. 인건비, 관리비 증가**
12. 전자메일을 선택적으로 내려받는 프로토콜은? → **4. POP3**
13. 통합 사무자동화 시스템 설명으로 알맞지 않은 것은? → **2. 복합화된 OA에서 단순 OA로 분리시키는 방향**
14. 사무자동화 추진의 선결 과제로 적합하지 않은 것은? → **4. 벤치마킹 실시**
15. 응용 프로그램에 가장 가까운 것은? → **1. 성적 처리 프로그램**
16. 계층형 데이터베이스에 대한 설명으로 가장 옳은 것은? → **2. 트리 구조 형식**
17. 정보처리 시스템의 변천 과정은? → **1. 비집중처리 → 집중처리 → 분산처리**
18. 자외선으로 내용을 지우고 다시 사용하는 ROM은? → **3. EPROM**
19. 사무자동화의 정량적 효과가 아닌 것은? → **3. 사무원들의 사기 진작에 따른 업무능률 향상**
20. 자료 송·수신기기가 아닌 것은? → **4. 전자출판**

## 2과목: 사무경영관리개론

### 핵심 정리

* 조도 기준: 정밀 작업은 300 lux 이상
* PERT: 일정 계획과 공정 분석에 유용
* DDC 600: 기술과학
* 정보보안 3요소: 기밀성, 무결성, 가용성
* 사무계획화: 목표를 정하고 효과적으로 수행하기 위한 계획

### 문제와 정답

21. 정밀 작업의 조도 기준은? → **2. 300 lux**
22. 사무관리와 정보관리의 관계 설명으로 올바른 것은? → **3. 정보관리는 의사결정에 필요한 광범위한 정보를 대상으로 한다**
23. PERT 기법의 장점으로 가장 옳지 않은 것은? → **3. 제한된 자원을 주공정과 상관없이 개별적으로 배치할 수 있다**
24. 경영정보 시스템의 기능별 분류로 적절하지 않은 것은? → **1. 운영통제정보 시스템**
25. 가상 모의실험을 의미하는 것은? → **4. Simulation**
26. EDIFACT의 3가지 기본 요소가 아닌 것은? → **4. 코드집**
27. 대실주의의 이점으로 가장 옳지 않은 것은? → **2. 상관의 감독을 어렵게 하며 그 범위를 좁힐 수 있다**
28. DDC에서 기술과학 코드는? → **3. 600**
29. 의사결정 시스템의 특성이 아닌 것은? → **4. 환경의 변화는 제외하고 설계되어야 한다**
30. Fayol의 관리 기능 범주에 속하지 않는 것은? → **4. Audit**
31. 전자기록물철의 분류번호는 어떻게 관리하는가? → **1. 등록 정보로 관리**
32. 정보보안의 3요소가 아닌 것은? → **2. 구속성**
33. 경영정보 시스템의 기능별 분류로 적절하지 않은 것은? → **1. 운영통제정보 시스템**
34. 사무실 조명 방법 중 성격이 다른 것은? → **4. 자연 조명**
35. 워크 샘플링법 설명으로 맞는 것은? → **3. 임의의 시간 간격으로 관측하여 시간적 구성비율을 통계적으로 추측하는 방법**
36. 경영관리단계 중 가장 먼저 실시해야 하는 것은? → **2. 계획화**
37. 문서 결재 설명으로 가장 옳지 않은 것은? → **4. 위임 전결 사항을 대결하는 경우 전결자란에 대결 표시**
38. 사무계획화를 가장 옳게 설명한 것은? → **2. 목표를 정한 후 효과적으로 수행하도록 하는 것**
39. CRM의 기대 효과와 가장 거리가 먼 것은? → **3. 고객의 수익 기여도와 무관**
40. 사무통제의 수단과 가장 거리가 먼 것은? → **2. Taylor System**

## 3과목: 프로그래밍 일반

### 핵심 정리

* 선점 스케줄링이 아닌 것: HRN
* 객체지향 언어 장점: 유지보수성, 재사용성
* 하향식 파서: Recursive Descent Parser
* 큐: 삽입과 삭제 위치가 다른 자료구조
* LRU: 가장 오랫동안 사용하지 않은 페이지 교체

### 문제와 정답

41. 선점 스케줄링 알고리즘이 아닌 것은? → **3. HRN**
42. 16K를 두 번째 공백 16K에 배치하는 전략은? → **3. Best Fit**
43. 객체지향 언어의 장점은? → **4. 유지보수성과 재사용성이 좋다**
44. BNF에서 정의를 나타내는 기호는? → **::=**
45. 코드 최적화 분류에 속하지 않는 것은? → **1. 색인 최적화**
46. 교착 상태 예방 방법으로 옳지 않은 것은? → **4. 환형 대기를 용인한다**
47. 하향식 파서에 해당하는 것은? → **3. Recursive Descent Parser**
48. 삽입은 한쪽, 삭제는 다른 쪽에서 일어나는 자료구조는? → **3. 큐**
49. 인터프리터 기법 설명으로 옳은 것은? → **2. 소프트웨어 시뮬레이션 방법으로 적절하다**
50. `a[index]=1+2;` 의 토큰 수는? → **4. 9**
51. 묵시적 제어 설명으로 적절한 것은? → **4. 프로그래머가 직접 제어하지 않는 경우 미리 정해진 순서대로 제어**
52. C 언어 do~while 문 설명 중 틀린 것은? → **1. 조건이 거짓인 동안 반복한다**
53. Type 1 문법을 인식하는 인식기는? → **3. Linear Bounded Automata**
54. 프로그램 수행 순서로 옳은 것은? → **3. 컴파일러 → 링커 → 로더**
55. 페이지 크기에 관한 설명으로 옳지 않은 것은? → **3. 페이지가 클수록 내부 단편화가 감소한다**
56. 시스템 타이머 만료나 콘솔 인터럽트 키 입력 시 발생하는 인터럽트는? → **2. 외부 인터럽트**
57. 문장 구조 중 성격이 다른 하나는? → **3. if(expression) statement-1; else statement-2;**
58. C 언어에서 생략되어서는 안 되는 것은? → **1. 함수 main**
59. 가장 오랫동안 사용하지 않은 페이지를 교체하는 기법은? → **4. LRU**
60. C 언어 관계 연산자 `!=`의 의미는? → **3. 같지 않다**

## 4과목: 정보통신 개론

### 핵심 정리

* 가상 회선 패킷 교환: 송신 순서대로 도착
* OSPF: Link State 방식 라우팅 프로토콜
* CODEC: 아날로그 음성과 디지털 간 변환
* 스트림 소켓: TCP 기반 통신
* ATM 셀 구조: Header 5 옥텟, Payload 48 옥텟

### 문제와 정답

61. 가상 회선 패킷 교환 방식 설명으로 옳은 것은? → **1. 송신된 순서대로 패킷이 도착한다**
62. 정보 통신 시스템 구성 요소 설명 중 옳지 않은 것은? → **1. 데이터 회선 종단장치 - DTE**
63. 시간 슬롯을 주기적으로 각 채널에 할당하는 다중화 방식은? → **3. 동기식 시분할 다중화기**
64. DSU의 역할은? → **1. 디지털 데이터를 디지털 신호로 변환시킨다**
65. OSI 7계층에 해당하지 않는 것은? → **4. Network Access Layer**
66. Link State 방식 라우팅 프로토콜은? → **4. OSPF**
67. ARQ 기법의 형태가 아닌 것은? → **3. Responsive-send ARQ**
68. 아날로그 음성 데이터를 디지털 형태로 변환·복원하는 것은? → **3. CODEC**
69. 시스템 회선 구성 방식이 아닌 것은? → **2. 비동기식 방식**
70. 감쇠량이 최소가 되는 조건은? → **2. LG = RC**
71. 프로토콜의 구성 요소가 아닌 것은? → **1. 재결합(Reassembly)**
72. 이동체 움직임에 따라 수신 주파수 세기가 변하는 현상은? → **4. 도플러 효과**
73. 프로토콜 계층화 장점이 아닌 것은? → **1. 전체적인 오버헤드가 증가한다**
74. TCP 전송 계층 프로토콜에 이용되는 소켓은? → **1. 스트림 소켓**
75. 패킷 교환망의 주요 기능과 거리가 먼 것은? → **3. 인터페이스**
76. TV 화면을 보며 상호대화 형태로 정보 검색을 하는 것은? → **2. Videotex**
77. ISDN 설명으로 잘못된 것은? → **2. 전송로가 아날로그이므로 중간에 모뎀이 사용된다**
78. 패리티 체크의 주된 목적은? → **2. 전송된 부호의 오류 검사**
79. HDLC 프레임 구조 순서로 옳은 것은? → **1. 플래그 → 주소부 → 제어부 → 정보부 → FCS → 플래그**
80. ATM 셀 구조로 옳은 것은? → **2. Header 5 옥텟, Payload 48 옥텟**

## 마무리

이번 회차는 데이터베이스, 프로그래밍, 정보통신 개론 전반의 기본 개념을 두루 묻는 문제가 많았다.
특히 암기만으로 풀기보다 개념 간 차이를 구분하는 문제가 자주 보여서, 단순 정답 암기보다 핵심 개념 정리가 더 중요하다고 느꼈다.

앞으로는 회차별 기출 정리와 함께 자주 틀리는 개념도 별도로 정리해볼 예정이다.
