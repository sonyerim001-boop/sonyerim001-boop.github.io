# 2025년 1회 사무자동화산업기사 필기 기출문제 정리

이번 글에서는 2025년 1회 사무자동화산업기사 필기 기출문제를 과목별로 정리했다.
정답만 나열하기보다 반복해서 등장하는 핵심 개념을 함께 확인할 수 있도록 구성했다.

---

## 1과목: 사무자동화 시스템

### 핵심 정리

* 운영체제는 하드웨어 자원을 관리하고 사용자와 시스템 사이의 인터페이스 역할을 한다.
* 빅데이터는 기존 방식으로 처리하기 어려운 대규모 데이터를 의미한다.
* B2B는 기업과 기업 간 전자상거래를 의미한다.
* RAID는 디스크 장애에 대비해 안정성을 높이는 기술이다.
* SSD는 반도체 기반 저장장치이다.
* 데이터베이스 모형에는 계층형, 네트워크형, 관계형 등이 있다.

### 문제와 정답

1. 운영체제에 관한 설명 중 가장 옳지 않은 것은? → **4. 응용 프로그램의 유지 보수를 담당하기 때문에 응용 소프트웨어라고도 한다**
2. 다음 중 입력장치에 해당하지 않는 것은? → **1. Plotter**
3. 다음 중 사무자동화의 기대 효과와 가장 거리가 먼 것은? → **3. 부수적 기능(Shadow Function)의 증가**
4. 팩시밀리에서 사진, 문자, 그림 등을 다수의 화소로 분해하는 과정은? → **3. 주사**
5. CALS의 개념에 포함되지 않는 것은? → **4. MPC(Multimedia Personal Computer)**
6. 방대한 데이터로 기존 방식으로는 처리하기 어려운 개념은? → **1. 빅 데이터**
7. 기업 간 전자상거래를 의미하는 용어는? → **2. B2B**
8. 집중 처리 시스템의 과도한 처리 집중에 따른 문제점이 아닌 것은? → **1. 통신관리상의 문제점**
9. 그룹웨어 시스템의 특징으로 옳지 않은 것은? → **3. 개인 작업만의 생산성 향상을 주목적으로 하고 있다**
10. S-ATA 인터페이스를 사용하고 반도체 기반으로 제작된 저장장치는? → **2. SSD**
11. 사무자동화 접근방법 유형에 속하지 않는 것은? → **3. 사원별 접근방법**
12. 내부·외부 비즈니스 데이터를 수집·가공해 관리자에게 정보를 제공하는 시스템은? → **2. MIS**
13. 사무자동화의 환경 개선 효과로 옳은 것은? → **3. 권태감 등의 감소로 작업자의 사기 고취**
14. 팩시밀리의 특징으로 틀린 것은? → **4. 동일 내용을 한 번에 한 명의 수신자에게만 보낼 수 있다**
15. 전자우편을 밀봉된 봉투 개념으로 보낸다는 개념은? → **1. PEM**
16. 디스크 장애에 대비해 안정성을 높이는 기술은? → **3. RAID**
17. 디스플레이나 프린터의 해상도 단위는? → **1. DPI**
18. 데이터베이스의 모형에 속하지 않는 것은? → **1. 정규형 데이터베이스**
19. 사무자동화로 기대되는 정성적 효과가 아닌 것은? → **3. 정보획득 시간이 길어진다**
20. 정보화시대 자동화 분류 설명으로 가장 바른 것은? → **3. 공장자동화 : 부분적인 자동화, 단위기계의 완전자동화, 생산 라인의 자동화 시스템**

## 2과목: 사무경영관리개론

### 핵심 정리

* 정보보안 3대 원칙은 기밀성, 무결성, 가용성이다.
* DDC에서 100은 철학, 300은 사회과학, 500은 자연과학에 해당한다.
* 전결은 결재권을 위임받은 자가 행하는 결재이다.
* 기능화의 원칙은 해야 할 일을 중심으로 조직을 구성하는 것이다.
* 빅데이터의 3V는 Volume, Velocity, Variety이다.

### 문제와 정답

21. 서식 설계 일반 원칙으로 가장 옳지 않은 것은? → **4. 특별한 사유가 없어도 별도의 기안문과 시행문을 작성한다**
22. 사무 표준화의 대상이 아닌 것은? → **4. 보안(Security)**
23. 처리 시기에 맞추어 담당자에게 서류를 전달하는 제도는? → **4. 자동독촉제도(Come Up System)**
24. 사무실 배치 원칙 설명으로 옳지 않은 것은? → **1. 각 사무실의 실내 배치를 우선 고려한다**
25. 사무분석 기법 중 사무공정 분석에 관한 내용이 아닌 것은? → **2. 사무작업 시간의 적정화**
26. 정보보안의 3대 원칙이 아닌 것은? → **4. 효율성**
27. 사무 간소화 대상과 가장 거리가 먼 것은? → **3. 정보 전달 등이 잘 안되는 1회성 작업**
28. 사무계획 수립 절차에 속하지 않는 것은? → **1. 시정조치**
29. 기록 양식과 기입 방법만 정확하게 관리되면 우수한 측정 방법은? → **4. 실적 기록법**
30. 사무조직화 일반 원칙에 가장 부합하지 않는 것은? → **1. 명령계통의 다원화**
31. 포름알데히드 관리 기준은? → **2. 0.1ppm 이하**
32. DDC 분류 연결이 틀린 것은? → **2. 200 : 기술**
33. 6월 사무관리 원가절감 목표 달성도는? → **3. 60%**
34. 정보통신망 구축 효과가 아닌 것은? → **4. 프로토콜의 다양화**
35. 결재권을 위임받은 자가 행하는 결재는? → **2. 전결**
36. 해야 할 일을 중심으로 조직을 구성해야 한다는 원칙은? → **3. 기능화의 원칙**
37. 정보통신서비스 제공자의 책무에 해당하지 않은 것은? → **4. 건전한 정보사회가 정착되도록 노력하여야 한다**
38. 빅데이터 3V와 가장 거리가 먼 것은? → **1. Vacant**
39. 안소프의 의사결정 유형이 아닌 것은? → **4. 혁신적 의사결정(Innovative)**
40. 사무관리 관리층 또는 사무관리자의 역할이 틀린 것은? → **4. 사무 관리자 - 부하직원 통제**

## 3과목: 프로그래밍 일반

### 핵심 정리

* 로더는 목적 파일을 메모리에 적재한다.
* 객체지향의 핵심 개념에는 추상화, 상속성, 자료 은폐 등이 있다.
* 어휘 분석은 토큰을 식별하는 단계이다.
* 디버깅은 논리적 오류를 찾아 수정하는 작업이다.
* 교착 상태 발생 조건에는 상호 배제, 점유와 대기, 비선점, 환형 대기가 있다.

### 문제와 정답

41. 실행 가능한 목적 파일을 메인 메모리에 적재하는 기능은? → **2. 로더**
42. 운영체제의 목적으로 거리가 먼 것은? → **4. 반환 시간 증대**
43. C 언어에서 나머지를 구하는 연산자는? → **1. %**
44. 객체지향 프로그램의 주요 개념이 아닌 것은? → **1. 고정 자료형(Fixed Data Type)**
45. 클래스 간 속성과 연산을 공유하는 객체지향 언어의 특징은? → **4. 상속성**
46. 어휘 분석 단계에 대한 설명으로 적절한 것은? → **4. 구문 단위를 형성하는 어휘 항목을 식별한다**
47. 기계어에 대한 설명으로 옳지 않은 것은? → **4. 모든 기계에서 공통으로 사용 가능하여 호환성이 높다**
48. EBNF에서 반복을 나타내는 기호는? → **4. { }**
49. Java에서 하위 클래스에서 상위 클래스를 참조하는 명령어는? → **3. Super**
50. 10진수 937의 BCD 코드는? → **1. 100100110111**
51. 특수 형태의 토큰에 해당되지 않는 것은? → **3. 상수**
52. 프로그램 안의 논리적 오류를 발견하고 수정하는 작업은? → **2. Debugging**
53. C 언어 기억 클래스에 해당하지 않은 것은? → **3. Internal**
54. “A＋B＊C－D”의 후위 표기법은? → **4. A B C ＊ ＋ D －**
55. 프로그래밍 언어에서 예약어란? → **3. 시스템이 알고 있는 특수한 기능을 수행하도록 이미 용도가 정해져 있는 단어**
56. carriage return을 나타내는 이스케이프 시퀀스는? → **4. \r**
57. C 언어에서 `char`는 어떤 변수 선언에 쓰이는가? → **3. 문자형**
58. 제시된 C 코드의 실행 결과는? → **1. A**
59. C 언어 연산자 중 우선순위가 가장 먼저인 것은? → **4. / (나눗셈 산술)**
60. 교착 상태 발생 조건이 아닌 것은? → **3. 선점**

## 4과목: 정보통신 개론

### 핵심 정리

* 데이터그램은 양이 적고 융통성이 요구되는 경우에 적합하다.
* 16진 QAM의 대역폭 효율은 4 bps/Hz이다.
* 100Base-T는 UTP 케이블을 사용한다.
* OSI 7계층의 네트워크 계층은 경로 선택 기능과 관련된다.
* DNS는 53번 포트를 사용한다.
* IEEE 802.11은 무선 랜 표준이다.

### 문제와 정답

61. 정보 통신 시스템의 특징이 아닌 것은? → **1. 정보 처리의 효율화**
62. 데이터 회선 종단장치와 관련이 없는 것은? → **2. DTE**
63. 정보 통신 시스템 처리 방식으로 틀린 것은? → **2. 주파수 분할 처리 방식**
64. 패킷망에서 데이터 양이 적고 융통성이 요구될 때 가장 적합한 교환방식은? → **4. 데이터그램(Datagram)**
65. 16진 QAM의 대역폭 효율은? → **2. 4 bps/Hz**
66. 100Base-T 네트워크에서 사용하는 전송 매체는? → **3. UTP cable**
67. PCM 변조 방식의 과정은? → **1. 표본화 → 양자화 → 부호화 → 복호화**
68. X 시리즈에서 패킷형 DTE와 DCE 간의 인터페이스는? → **4. X.25**
69. L2 스위치의 기본 기능이 아닌 것은? → **4. Routing**
70. 전송 과정 오류의 주요 원인이 아닌 것은? → **4. 패리티 비트**
71. 무잡음 이산 채널의 채널 용량은? → **2. C = log2(n)**
72. OSI 7계층의 네트워크 계층 기능과 관계가 있는 것은? → **1. 경로 선택**
73. 9,600bps 데이터를 8진 PSK로 변조할 때 변조 속도는? → **3. 3,200 Baud**
74. 정보의 전달 체계를 무엇이라 하는가? → **2. 정보 통신망**
75. DNS 서버가 사용하는 TCP/UDP 포트 번호는? → **3. 53**
76. CSMA/CD에 대한 내용으로 잘못된 것은? → **1. 모든 제어기는 차등의 액세스 권리를 갖는다**
77. 무선 랜의 표준 규격은? → **3. IEEE 802.11**
78. 통신 소프트웨어의 세 가지 기본 구성요소는? → **1. 데이터 송·수신, 통신 하드웨어 제어, 이용자 인터페이스 제어**
79. Shannon 표본화 정리에 따르면 최고 주파수의 최소 몇 배 이상으로 표본화해야 하는가? → **2. 2배**
80. 디지털 데이터를 아날로그 신호로 변환시키는 키잉 방식에 해당하지 않는 것은? → **1. 스펙트럼 편이 키잉**

## 마무리

이번 회차는 사무자동화, 사무경영관리, 프로그래밍, 정보통신의 기초 개념을 폭넓게 묻는 문제가 많았다.
특히 단순 암기보다는 용어의 정확한 의미와 유사 개념 간 차이를 구분하는 문제가 많아서, 개념 정리 중심의 복습이 중요하다고 느꼈다.

앞으로도 회차별 기출문제를 정리하면서 자주 헷갈리는 개념을 함께 정리해 나갈 예정이다.
